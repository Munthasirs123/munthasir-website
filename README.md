# Personal Website of Munthasir Shiraz

This is the source code for my personal website and blog, built with a modern, content-first architecture.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Content**: Markdown (`.md`) files
- **Fonts**: [Inter](https://fonts.google.com/specimen/Inter) and [Lora](https://fonts.google.com/specimen/Lora)
- **Deployment**: [Vercel](https://vercel.com)
